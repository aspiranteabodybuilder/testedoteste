💪 LISTA DE COMANDOS NO CHAT DO TWITCH.TV/ASPIRANTETV 💪

 

 

!salve - comando que manda um salve para os usuarios no chat.
 !youtube - comando com o link do canal no youtube do aspirante.
 !sr - comando que,ao usado com um  link do youtube de uma musica,adiciona a mesma no nightbot.
 !paladins - mostra os todos os comandos do paladins que podem ser usados no chat.
 !intervalo - comando usado pelo mod quando o streamer está em intervalo.
 !follow - comando que informa que temos um novo seguidor no canal.
 !uptime - tempo de live corrido.
 !pc - especificações do pc da stream
 !frango - use o comando marcando alguem do chat e o chame de frango.
 !pistola - comando usado quando o aspirante pistola na live.
 !ban - marque alguém logo depois desse comando e dê um ban na pessoa marcada.
 !seguidores - total de seguidores do canal.
 !tempo - mostra o tempo que o seguidor,está seguindo o canal. 
 !café - comando usado sempre que o aspirante estiver com um café na mão.
 !pragmatico - comando usado para declarar ao chat que voce considerou a atual musica sendo tocada,muito pragmatica.
 !coerapaziada - manda um COE RAPAZIADA no chat(audio)
 !mod - comando usado por moderadores para informar que há moderadores online no chat.
 !gankmittow - clip do dia que o mittow gankou o canal.
 !doacao - acesse o link para fazer sua doação e contribuir conosco.
 !hirez - comando usado sempre que o Paladins(crasha)
 !loja - acessar a loja do canal.
 !itens - saber quais itens estão disponíveis na loja.
 !redeem - comando que resgata um item da loja a partir do chat,sintaxe do comando: !redeem  "nomedoitem",Exemplo: !redeem rankeadacomaspirante
 !bateu - comando suado sempre que o aspirante bate a carreta no eurotruck simulator.
 !trans - comando para informar que a live ta uma pourra kkk
 !pedrada - comando que informa que a musica que começará a tocar é muito boa.
 !pontos - saiba quantos pontos voce tem no canal(aspirantecoins)
 !jogademais - manda um áudio pra live dizendo que o aspirante fez uma bela jogada no game.
 !soco - use o comando marcando uma pessoa do chat e dê um soco nela

 

 

💪 PARA USO DOS MODERADORES 💪
 
 !sraffle: abrir a rifa / !intervalo: usado quando o aspirante está em intervalo / !live: audio que informa que a live já está começando / !demon: marcando uma pessoa,manda um audio dizendo pro usuario nao esquecer de seguir o canal. / !follow: manda um audio dizendo que temos um novo seguidor no canal / !ajuda : envia um audio para o aspirante informando que estão precisando de ajuda no chat.
